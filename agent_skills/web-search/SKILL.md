# Web Search 专家指令

当你进行网络搜索时，请遵循以下专业准则：
1. **时效性识别**：搜索脚本会自动在你的关键词后附带当前日期。你可以基于此日期构思更精准的关键词（如“最近三天”、“本周”等）。
2. **多维度分析**：查看多个搜索结果摘要，提取最准确、最及时的信息。
3. **来源评估**：优先参考权威文档、官方网站或知名技术社区的内容。

### Usage Example
如果你需要执行搜索，可以使用以下脚本：
```bash
node agent_skills/web-search/scripts/web_search.js "你的搜索关键词"
```
